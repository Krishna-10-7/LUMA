React

React-like Reactivity

Functional components, hooks (useState, useEffect, useRef, useMemo), and virtual DOM rendering.

const App = () => { const [count, setCount] = useState(0); return h('button', { onClick: () => setCount(count + 1) }, count ); };
Animate

GSAP-style Animations

30+ easing functions, timelines, stagger, scroll triggers, parallax, and 20+ preset animations.

LumaAnimate.to('.box', { transform: { translateX: 200 }, backgroundColor: '#ff6b6b' }, { duration: 1000, easing: 'easeOutBack' });

Get Started in Seconds

No build tools required. Just include and use.

<!-- Include LumaJS --> <script src="luma.umd.min.js"></script> <script src="luma-animate.min.js"></script> <script src="luma-3d.min.js"></script> <script> // Use React-like components const App = () => { const [count, setCount] = Luma.useState(0); return Luma.h('div', {}, Luma.h('h1', {}, count), Luma.h('button', { onClick: () => setCount(count + 1) }, 'Increment') ); }; // Animate anything LumaAnimate.to('.box', { transform: { translateX: 200 } }, { duration: 1000 }); // Create 3D scenes const scene = new Luma3D.Scene(); const cube = new Luma3D.Mesh( new Luma3D.BoxGeometry(1,1,1), new Luma3D.Material({color: [1,0.3,0.3]}) ); scene.add(cube); </script>
